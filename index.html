// About the Project

What is AI-IDS?

AI-IDS is a locally-running intrusion detection system that detects zero-day cyber attacks without relying on any predefined attack signatures. It combines Isolation Forest unsupervised ML, LLM-based reasoning, and automated tiered response in a single Python terminal application.

🤖
Isolation Forest ML
Unsupervised anomaly detection — learns normal traffic behaviour and flags deviations. No labeled attack data needed.
🧠
LLM Reasoning Engine
Ollama / Llama 3 generates human-readable threat analysis. Built-in expert rule engine as a zero-setup fallback.
⚡
Attack Simulation
6 built-in attack profiles — Port Scan, DDoS, Data Exfiltration, Brute Force, C2 Beacon, NULL Scan.
🛡️
Auto Response Engine
Four-tier automated response: LOG → THROTTLE → BLOCK IP → ISOLATE SEGMENT based on severity score.
📊
Live Terminal Dashboard
Full-screen Rich terminal UI with sparklines, alert feed, severity charts, and protocol breakdown — in real time.
🔄
Online Retraining
Model retrains on accumulated traffic data mid-session. Continuously improves detection accuracy over time.
